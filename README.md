# 🧶 منصة خيوط الفريدة للكروشيه | لوحة التحكم والمتجر الإلكتروني

منصة إلكترونية متكاملة ومخصصة لبراند **خيوط الفريدة للكروشيه**، تم تصميمها وتطويرها لتوفير تجربة مستخدم راقية وبسيطة للزوار، مع لوحة تحكم ذكية وشاملة لإدارة المخزن والمبيعات بكل سهولة.

---

## 🚀 مميزات المنصة

### 1. المتجر الإلكتروني العام (`index.html`)
* **تصميم باستيل راقٍ (Pastel Minimalist):** واجهة مريحة للعين ومتناسقة بالكامل مع الهوية البصرية الدافئة للبراند.
* **عرض ديناميكي للمنتجات:** يتم جلب المنتجات تلقائياً من قاعدة البيانات مع إخفاء القطع غير المتوفرة فوراً.
* **فلو طلب سريع عبر الواتساب (Ultra-Fast Checkout):** إمكانية طلب المنتج بضغطة زر واحدة تقوم بإنشاء رسالة تلقائية ومجفرة باسم المنتج المختار وإرسالها مباشرة لواتساب المحل.
* **معلومات البراند بالكامل:** يحتوي تذييل الصفحة (Footer) على مواعيد العمل، وعنوان المحل في الإسكندرية، ونبذة عن قصة البراند لبناء الثقة مع العملاء.

### 2. لوحة تحكم الأدمن (`admin.html`)
* **تسجيل دخول آمن بجوجل (Google OAuth):** حماية كاملة عبر Supabase Auth لمنع أي دخول غير مصرح به.
* **نظام حظر صارم:** يتم التحقق من البريد الإلكتروني للمستخدم فور تسجيل الدخول، ولا يتم السماح إلا للإيميل الرسمي للمحل بالوصول للوحة التحكم، مع طرد أي حساب آخر تلقائياً.
* **إحصائيات حية (Live Dashboard):** كروت إحصائية ذكية تعرض إجمالي المنتجات، القطع المتاحة فوري، والقطع غير المتوفرة في الوقت الفعلي.
* **إدارة المنتجات (CRUD):** إضافة منتجات جديدة، تعديل المنتجات الحالية، ورفع الصور مباشرة إلى السيرفر مع ميزة المعاينة الفورية قبل النشر.
* **التحكم السريع في المخزن:** زر تبديل سريع (Toggle Switch) لتغيير حالة المنتج (متوفر / غير متوفر) مباشرة من الجدول مع نافذة تأكيد لمنع الأخطاء.
* **كود المخزن السري (`storage_code`):** حقل مخصص يظهر للأدمن فقط لتحديد مكان حفظ القطعة أو الرف السري الخاص بها في المخزن الحقيقي.

## 🛠️ التقنيات المستخدمة

* **Frontend:** HTML5, CSS3 (Flexbox & CSS Grid), Vanilla JavaScript.
* **Database & Auth:** Supabase (PostgreSQL).
* **Styling & Fonts:** Google Fonts (Cairo / Tajawal), Font Awesome / Lucide Icons.
* **Deployment:** GitHub Pages / Netlify / Google Cloud Console.

## 🗄️ هيكلة قاعدة البيانات (Supabase Table Schema)

الجدول الأساسي للمنتجات يحمل اسم `public.products` ويحتوي على العواميد التالية بالملي:

| اسم العمود | نوع البيانات | الوصف |
| :--- | :--- | :--- |
| `id` | `int8` (Identity) | الرقم التعريفي التلقائي للمنتج (يتم إنشاؤه تلقائياً بواسطة الداتابيز). |
| `title` | `text` | اسم منتج الكروشيه. |
| `price` | `numeric` | سعر المنتج. |
| `image` | `text` | رابط رابط الصورة المرفوعة على الـ Bucket. |
| `category` | `text` | القسم التابع له المنتج (خيوط، سنارات، إكسسوارات... إلخ). |
| `description` | `text` | وصف تفصيلي للمنتج. |
| `storage_code` | `text` | كود الرف أو المخزن السري (للأدمن فقط). |
| `is_in_stock` | `boolean` | حالة توفر المنتج (True = متوفر فوري / False = غير متوفر). |
| `created_at` | `timestamptz` | وقت وتاريخ إضافة المنتج تلقائياً. |
